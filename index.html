Deskripsi Sistem

💻 Sistem Digital

Sate Nusantara adalah sistem berbasis digital yang memudahkan pengguna dalam mengakses informasi menu secara online.

📋 Pemesanan Mudah

Pengguna dapat melihat menu dan melakukan pemesanan dengan cepat tanpa proses yang rumit.

⚙️ Manajemen Data

Admin dan kasir dapat mengelola menu serta memproses transaksi secara terstruktur dan efisien.

Sistem ini dirancang untuk memberikan kemudahan dan kenyamanan bagi pengguna.

Identifikasi Aktor

👤 Pelanggan

Aktor yang menggunakan sistem untuk melihat menu, melakukan pemesanan, dan melakukan pembayaran.

🛠️ Admin

Bertugas mengelola data menu seperti menambah, mengubah, dan menghapus menu dalam sistem.

💳 Kasir

Bertanggung jawab dalam memproses transaksi, menerima pembayaran, dan mencetak struk.

Ketiga aktor ini saling berinteraksi dalam sistem untuk memastikan proses berjalan dengan baik.

Use Case

  • Melihat Menu
  • Memilih Makanan
  • Melakukan Pemesanan
  • Melakukan Pembayaran
  • Menerima Struk

🛠️ Admin

  • Menambah Menu
  • Mengubah Menu

  • Mengelola Data

💳 Kasir

  • Menerima Pesanan
  • Memproses Pembayaran
  • Mencetak Struk
  • Mengonfirmasi Transaksi

Use case ini menggambarkan interaksi antara aktor dan sistem dalam proses pemesanan sate.

Relasi Sistem

👤 User → Sistem

User berinteraksi dengan sistem untuk melihat menu, memilih makanan, dan melakukan pemesanan.

💳 Kasir → Sistem

Kasir memproses pembayaran, mengonfirmasi pesanan, dan mencetak struk melalui sistem.

🛠️ Admin → Sistem

Admin mengelola data menu dan memastikan sistem berjalan dengan baik dan terstruktur.

Relasi ini menunjukkan bagaimana setiap aktor berinteraksi dengan sistem dalam proses pemesanan sate.
